關於我們

空間與環境

大宏廷首先注重的，是您的所在區域環境。其包含以下的要素：

  1. 立地條件、物理環境、基地位置等客觀的要件
  2. 您與環境之間互動的關係
  3. 您的鄰居、鄰里關係
  4. 如何善用所在環境的條件及優勢
  5. 如何避免所在環境的缺點並加以改善

空間與人

使用空間的是人，大宏廷深入了解每一個吹毛求疵、特立獨行的您、家人或工作團隊，考慮空間與所有成員之間的關係。其中包含：

  1. 生活、工作的習慣與動線
  2. 空間的劃分與利用，且有效的利用
  3. 櫥櫃機能及收納的規劃
  4. 傢俱、傢俬的設計及規劃
  5. 藝術、視覺美學的規劃

空間與時間

大部分的室內設計者都以現階段的需求做規劃，但是大宏廷提出時間的觀念。其中包含：

  1. 未來三～五年的成員變化
  2. 未來三～五年功能需求的變化
  3. 您的物件未來的價值在哪？
  4. 如何幫您去提升物件未來的價值？
